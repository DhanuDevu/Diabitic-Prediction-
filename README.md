# Diabetes Prediction Project
Overview

This project predicts whether a person has diabetes or not using machine learning. It uses medical data like glucose level, BMI, age, etc. to make predictions.   

## Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib

## Dataset
Pima Indians Diabetes Dataset
File used: diabetes.csv
      
## How to Run
In Google Colab
Upload the .ipynb file
Upload diabetes.csv
Click Run All
